# 🌐 BFS & DFS Graph Traversal


A Python implementation of **Breadth First Search (BFS)** and **Depth First Search (DFS)** algorithms using graph traversal.


## 📌 About The Project


This project was developed as part of my engineering coursework to understand graph data structures and traversal techniques.


The program demonstrates how BFS and DFS algorithms explore nodes in a graph.


This project helped me practice:

- Graph representation
- Queue data structure
- Recursion
- Sets
- Algorithm implementation
- Problem solving



## 🚀 Features


✅ Breadth First Search traversal  
✅ Depth First Search traversal  
✅ Graph representation using dictionary  
✅ Simple command-line execution  


## 🛠️ Technologies Used


- Python 3
- Collections module
